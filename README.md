# film-time
영화제 시간표 작성
🤖 이 프로젝트는 Anthropic의 Claude로 작성되었습니다.

❓ 영화제별로 보고 싶은 영화를 등록하고, 시간이 겹치지 않게 나만의 시간표를 짜는 웹 앱입니다.

🔗 https://ruiyamanobe.github.io/film-time/film-time.html 

❤️ 주요 기능

- 여러 영화제를 등록하고 따로 관리
- 보고 싶은 영화를 목록에 등록 (제목 / 시간 / 상영관 / 메모)
- 시간이 겹치는 영화는 시간표에 추가가 막히고, 어떤 영화와 겹치는지 바로 표시됨
- 영화제 전체 기간 중 실제로 가는 날만 골라서 시간표에 표시
- 아이폰 홈 화면에 추가하면 앱처럼 사용 가능

❗️ 데이터 저장 및 개인정보

❶ 이 앱은 입력한 데이터를 브라우저의 `localStorage`에 저장합니다. 흔히 말하는 쿠키와 비슷하게 "내 브라우저에 로컬로 저장된다"는 개념은 같지만, 쿠키처럼 서버로 전송되지는 않습니다.

❷ 다른 사람과 공유되지 않습니다. 같은 주소를 다른 사람이 열어도 그 사람 브라우저에만 데이터가 쌓이며, 서로의 시간표는 보이지 않습니다.

❸ 앱을 닫거나 기기를 재시작해도 데이터는 유지됩니다. 사용자가 직접 "사이트 데이터 삭제"를 하거나 시크릿/비공개 모드로 사용한 경우에만 사라집니다.

❹ 아이폰 사용 시 주의할 점
사파리에는 추적 방지 기능(ITP)이 있어서, 일반 사파리 탭으로 열어본 사이트를 7일간 한 번도 방문하지 않으면 해당 사이트의 `localStorage`를 자동으로 삭제합니다. 이 앱이 잘못 동작하는 게 아니라 사파리의 개인정보 보호 정책 때문입니다. 이를 피하려면 사파리 공유 버튼 → ‼️‼️‼️홈 화면에 추가‼️‼️‼️ 로 설치해서 사용하는 것을 권장합니다. 홈 화면에 추가된 앱은 사파리와 별도로 취급되어 이 7일 제한에서 완전히 제외됩니다. PC나 안드로이드 크롬에는 이런 제한이 없습니다.

❺ 기기/브라우저 간 데이터 동기화는 지원하지 않습니다. 각자 자기 기기에만 저장됩니다.
